Class TFBuffer

Holds a block of data, suitable to pass, or retrieve from TensorFlow.

Remarks

Use the TFBuffer to blobs of data into TensorFlow, or to retrieve blocks of data out of TensorFlow.

There are two constructors to wrap existing data, one to wrap blocks that are pointed to by an IntPtr and one that takes a byte array that we want to wrap.

The empty constructor can be used to create a new TFBuffer that can be populated by the TensorFlow library and returned to user code.

Typically, the data consists of a serialized protocol buffer, but other data may also be held in a buffer.
