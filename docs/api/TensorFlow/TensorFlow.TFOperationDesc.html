public class TFOperationDesc : TensorFlow.TFDisposable
Remarks

This is the low-level API that is used to create operations by manually specificying all the parameters of an operation (inputs, outputs, attribute descriptions) that can then be attached into a graph.

Generally, you will instead be using the methods surfaced in TFGraph that surfaces a C# high-level API that has already been bound to the built-in TensorFlow nodes.

You create instances bound to a graph, add inputs, attributes and so on, and when you are done you can call the FinishOperation(TFStatus) method that will turn this TFOperationDesc into a TFOperation.

Remarks

A control input is an Operation that must be executed before running the operation currently being built.

For example, an Assert operation may be added as a control input for this operation. The Assert now behaves as a pre-condition that will always verify itself before running the operation.
