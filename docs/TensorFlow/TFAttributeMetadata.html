TFAttributeMetadata Struct

Low-level: this describes the tensorflow type information for an attribute in the low-level attributes used by operations.

Syntax

public struct TFAttributeMetadata

Remarks

This is a low-level operation returned by the TFOperation.GetAttributeMetadata. This is included for completeness, but is not generally used from C#, as you have access to the high-level bindings in the TensorFlow.TFGraph type.
