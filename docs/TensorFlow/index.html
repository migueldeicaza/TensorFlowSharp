TensorFlow Namespace

Namespace

This is the API documentation for the TensorFlowSharp project.

Type Description
TFAttributeMetadata Low-level: this describes the tensorflow type information for an attribute in the low-level attributes used by operations.
TFAttributeType Low-level: Enumeration describing the types of a metadata attribute
TFBuffer Holds a block of data, suitable to pass, or retrieve from TensorFlow.
TFBuffer.BufferReleaseFunc Signature of the method that is invoked to release the data.
TFCode Status code for invoking a tensorflow operation.
TFCore Contains TensorFlow fundamental methods and utility functions.
TFDataType The data type for a specific tensor.
TFDisposable Base class for many TensorFlow data types that provides a common idiom to dispose and release resources associated with the native data types. Generally, you do not need to use this.
TFException TensorFlow Exception
TFGraph Represents a computation graph. Graphs may be shared between sessions and are thread safe.
TFGraph.WhileConstructor Signature of the method that will be invoked by the TFGraph.While method to construct a while loop
TFImportGraphDefOptions Contains options that are used to control how graph importing works.
TFInput Documentation for this section has not yet been entered.
TFLibrary Documentation for this section has not yet been entered.
TFOperation Represents a computation node in the graph. Tensorflow operations are attached to a Tensorflow.TFGraph.
TFOperationDesc Low-level TensorFlow operation builder
TFOutput Represents a specific output of an operation on a tensor.
TFScope TFGraph name scope handle
TFSession Drives the execution of a graph
TFSession.PartialRunToken Token returned from using one of the Partial Run Setup methods from TensorFlow.TFSession, and use this token subsequently for other invocations.
TFSession.Runner Use the runner class to easily configure inputs, outputs and targets to be passed to the session runner.
TFSessionOptions Documentation for this section has not yet been entered.
TFShape Represents the shape of a tensor
TFStatus Used to track the result of TensorFlow operations.
TFTensor TFTensor holds a multi-dimensional array of elements of a single data type.
TFTensor.Deallocator Signature that methods must conform to to be used to release memory that was passed to a manually allocated TFTensor
